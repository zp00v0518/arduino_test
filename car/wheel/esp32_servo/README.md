# 📑 Технічна інструкція: Система керування кермом (ESP32 + OK2D86ECS)

Ця інструкція описує фінальну конфігурацію системи з акцентом на надійність та безпеку (Fail-Safe).  
Система використовує сигнал **Alarm** для моніторингу працездатності драйвера та механічний **Self-Test** при старті.

---

## 1. 🔌 Схема підключення (Wiring Diagram)

Для роботи системи необхідно **2 резистори по 10 кОм**.

| Пін ESP32 | Пін драйвера | Колір / Примітка | Опис |
|----------|-------------|------------------|------|
| GND | COM- / GND | Спільний | Спільна "земля" (мінус) для ESP32 та драйвера |
| 3.3V | PUL+ / DIR+ / ALM+ | Перемичка | Живлення оптопар (3.3В від ESP32) |
| GPIO 12 | PUL- (Step) | Сигнал | Команди кроків (частота визначає швидкість) |
| GPIO 13 | DIR- (Dir) | Сигнал | Напрямок повороту (HIGH/LOW) |
| GPIO 14 | ALM- (Alarm) | Safety | Вхід тривоги. Обов'язково резистор 10к на GND |

### ❓ Чому немає Pend?

Сигнал **ALM (Alarm)** є пріоритетним.  
Він сигналізує про критичні помилки:
- обрив енкодера  
- перегрів  
- втрата живлення  

У режимі реального часу для авто цього достатньо для безпечної зупинки.

---

## 2. ⚙️ Конфігурація драйвера (Parameter Settings)

Налаштуйте параметри за допомогою кнопок на передній панелі драйвера.  
Після зміни **затисніть SET для збереження**.

| Параметр | Значення | Опис |
|----------|---------|------|
| P-001 | 1600 | Мікрокрок (Microstep). 1600 імпульсів на 1 оберт вала |
| P-080 | 0 | Режим керування: Pulse + Direction (Крок + Напрямок) |
| P-100 | 1 | Логіка ALM: Normally Closed (сигнал є, коли все ОК) |
| P-110 | 1 | Функція ALM: Вихід тривоги (Alarm) |

> ⚠️ Після налаштування обов’язково вимкніть та ввімкніть живлення **48В / 220В**

---

## 3. 📡 Логіка статусів телеметрії

Система передає на сервер коди стану для миттєвої діагностики:

- **Status 0 (OK)**  
  Драйвер працює, помилок немає

- **Status 1 (Alarm)**  
  Драйвер вимкнено, дріт відпав або сталася критична аварія мотора

- **Status 2 (Test Fail)**  
  Механічний тест при старті не пройшов (кермо не змогло смикнутися)
